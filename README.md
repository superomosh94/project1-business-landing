# Derivo Tech Solutions - Landing Page

A professional landing page for a web development service targeting Kenyan businesses.

## Features
- Fully responsive design (mobile-first)
- Fast loading (optimized images and code)
- Contact form with validation
- Service pricing display
- Portfolio showcase
- WhatsApp integration
- SEO optimized

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- Font Awesome icons

## Pages/Sections
1. Navigation Bar
2. Hero Section
3. Services Offered
4. Pricing Table
5. Portfolio Showcase
6. Contact Form
7. Footer

## Quick Start
1. Download all files into one folder
2. Replace placeholder images in `/images/`
3. Update contact information in `index.html`
4. Update WhatsApp number in `script.js`
5. Open `index.html` in a browser

## Customization
- Change colors in `:root` variables in `style.css`
- Update services and prices in `index.html`
- Add your own portfolio projects
- Connect contact form to your backend
